Calcolo Orari — PWA con icona personalizzata

Pubblicazione su GitHub Pages:
1) Carica tutti i file di questa cartella nel repository (root):
   - index.html
   - manifest.webmanifest
   - sw.js
   - /icons (3 PNG)

2) GitHub → Settings → Pages → Deploy from a branch → main / (root)

Aggiornamento icona su iPhone:
- Se l'app era già presente in Home: elimina l'icona dalla Home.
- Apri l'URL in Safari, poi Condividi → Aggiungi a Home.
